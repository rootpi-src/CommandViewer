# CommandViewer

This plugin is based on a Paper bukkit.

Display commands in actionbar when you facing command block with diamond sword.

Run commands when you sneaking and click command block with diamond sword.

## Tested Version

- 1.16.2

- 1.15.2

- 1.14.4

- 1.13.2
